# 什么是多元记

你的手机中是否有一堆图片，既不想整理，也不肯删除？  
你的设备中是否有一堆文件，散落在各个文件夹中，想用时却找不到？  
数据格式多种多样，你想把文字、图片、视频等归纳到一起？  
数据要用到时，却后悔设备没带在身上？  
你是否纠结于各种网盘巨慢的下载速度？  
你是否想记录生活，却找不到合适的工具？  

那么**多元记**就是为你而生。

## 多元级做了什么

多元记可以记录各种数据类型，将其绑定为条记录，并默认展示为一条时间线。

- 支持多种数据类型，包括文本、图片、视频、坐标、标签、文件、人员等。
- 对不同的数据类型提供了不同的展示方式，也提供了各种记录视图。
- 前沿的设计：毛玻璃、明暗主题切换、模块化扁平化设计。
- 方便的管理数据：对各种数据类型做了不同的筛选处理。
- 快捷的记录分享：筛选出来的数据可以很方便分享到各种社交平台。
- 数据安全：数据存储在云端，访问读写都需要令牌授权，大可放心
- 用户隐私保护：作者在这里保证，不会在未经用户同意的情况下，用隐私数据做任何事情

## 作者的话

这个项目是我大学期间开始的，当时在学 SpringBoot 和 Vue 的开发，边学边做。

最初只是做了一个校园墙整合网，主要是通过爬虫来获取QQ空间校园墙的数据，
然后发送到后端服务器去做一个整合，最后展示为现在的首页的那些公开数据。  

但是后来我发现手机和电脑上数据太多了，而且也乱，我这人又健忘，也喜欢拍照，偏偏还是个处女座有洁癖。
所有想做一个记录网站去管理数据。

后来我发现生活中很多数据是有强绑定关系的，尽管是不同类型的数据，
比如你上课时对老师课件拍的照片和你记在手机上的笔记，这照片和笔记就会有一种很紧密的关系。
而且大部分数据都和时间有关系，

因此我做了一个可以把各种类型数据绑定为一条记录的网站，默认以时间线展示。

## 收费

本网站服务器运维、前后端开发、数据存储都是我一个人做的，目前都是作者用爱发电QAQ。  
未来如果用户量多起来了，可能会考虑收费（或者加点广告啥的？），当然我会尽量保证这个网站是免费的。  
目前最烧钱的就是腾讯云的对象存储的服务，其中存储空间和下行流量费是最贵的，如果收费，那大部分给的是这个钱。
未来我会把这些成本公布出来，保证向大众的公开透明。
